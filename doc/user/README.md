# Codeenstein 3D — Player Guide

Codeenstein 3D turns a folder of source code into a playable retro raycaster dungeon: every folder is a level, every file is a room, and every function is an enemy. This guide covers everything you need to actually play — for the project pitch and dev setup, see the top-level [`README.md`](../../README.md).

- [Tips](tips.md) — practical advice for staying alive
